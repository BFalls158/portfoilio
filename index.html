Book Buddies

An AngularJS app that facilitates communication for trading books!

New

Book Buddies was my final project at Grand Circus and the application that was presented at Grand Circus' Demo Day. It was designed and coded by myself, Jennifer Nsaaz, Marcell Hayes, and Emily Nelson.

Book Buddies is a single page, AngularJS app built on an Express server, using a PostgreSQL database. It utilizes the Google Books API to populate books for users to add to their libraries and watchlists, which the application then uses to match their book interests with other users. They can then contact the users through our system, which uses the MailJet API to facilitate communcation between the users.
